# SaaS Pricing Page (Production UI)

## 🚀 Live Demo
https://your-live-link.vercel.app

## 📌 Overview
This project is a fully responsive SaaS pricing page designed with a focus on modern UI/UX principles and conversion optimization.

## ✨ Features
- Monthly ↔ Yearly pricing toggle
- Dark mode (with localStorage persistence)
- Responsive design (mobile-first)
- Highlighted "Most Popular" plan
- Interactive FAQ accordion
- Smooth micro-interactions (hover, transitions)

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 Approach
- Built a clean and minimal UI using CSS variables
- Focused on accessibility and responsiveness
- Implemented reusable components like pricing cards
- Added interactive elements to improve user engagement

## ⚡ Challenges
- Managing state for pricing toggle
- Creating smooth FAQ animations
- Ensuring consistent UI in dark mode

## 🚀 Improvements (Future Scope)
- Add authentication (login/signup)
- Integrate payment gateway (Stripe)
- Convert into React + Tailwind project
- Add animations using Framer Motion
